## ✨ الهامي خبرې

"کتاب د انسان تر ټولو وفادار ملګری دی،  
چې نه دروغ وایي، نه ستړی کېږي، او نه بې لارې کوي.  
راځه چې د مطالعې له لارې خپل فکرونه روښانه کړو  
او د پښتو ژبې خزانه خپلو راتلونکو نسلونو ته وساتو."
